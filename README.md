# Sistema de Pagos 🎓💳

Proyecto fullstack desarrollado con **Spring Boot (backend)** y **Angular (frontend)** para la gestión de estudiantes y pagos.  
Incluye documentación automática con **Swagger UI** y arquitectura modular lista para escalar.

## 🚀 Tecnologías utilizadas
- Backend: Java 17, Spring Boot, JPA/Hibernate, Swagger/OpenAPI
- Frontend: Angular 17, TypeScript, Angular Material
- Base de datos: MySQL

## 📂 Estructura del proyecto

| Carpeta / Archivo          | Descripción                                |
|-----------------------------|--------------------------------------------|
| `backend/`                 | API REST con Spring Boot                   |
| `sistema-pagos-frontend/`  | Aplicación Angular                         |
| `.gitignore`               | Configuración para ignorar archivos en Git |
| `README.md`                | Documentación principal del proyecto       |

## ⚙️ Instalación y ejecución

### Backend
```bash
cd backend
mvn spring-boot:run
```
Acceder a Swagger UI http://localhost:8080/swagger-ui/index.html

## Frontend

```Bash
cd sistema-pagos-frontend
npm install
ng serve
```
Acceder en navegador: http://localhost:4200
