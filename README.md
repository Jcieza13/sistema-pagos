\# Sistema de Pagos



Proyecto fullstack que integra \*\*backend en Java (Spring Boot)\*\* y \*\*frontend en Angular\*\* para la gestión de estudiantes y pagos.



\## 🚀 Tecnologías

\- Backend: Java, Spring Boot, Maven

\- Frontend: Angular 17

\- Base de datos: MySQL

\- Documentación: Swagger



\## 📂 Estructura del Proyecto



| Carpeta/Archivo | Descripción                          |

|-----------------|--------------------------------------|

| `sistemapagos/` | Raíz del monorepo del sistema de pagos |

| └── `backend/`  | Módulo backend en Java (Spring Boot) |





\## ⚙️ Instalación y ejecución



\## ▶️ Cómo ejecutar

1\. Clonar el repositorio:

&nbsp;  ```bash

&nbsp;  git clone https://github.com/Jcieza13/sistema-pagos.git

&nbsp;  ```





\##  2. Backend

```bash

cd backend

mvn spring-boot:run

```

Acceder a Swagger UI: http://localhost:8080/swagger-ui/index.html



\## 3. Frontend 

```bash

cd sistema-pagos-frontend

npm install

ng serve

```



Acceder en Navegador: http://localhost:4200



\## 📖 Documentación



La API del backend está documentada con \*\*Swagger\*\*.  

Una vez levantado el servidor, puedes acceder a la documentación interactiva en:

http://localhost:8080/swagger-ui.html



Esta documentación incluye:

\- Endpoints disponibles

\- Métodos HTTP soportados

\- Modelos de datos
